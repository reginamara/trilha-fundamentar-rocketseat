# Tipos de Dados

Conforme o ECMAScript standard temos 9 tipos de dados:

* Data types
    * Primitive / Primitive value
    * Structural
    * Structual Primitieve

## Primitivos - Não sofrem alteração

* String
* Number
* Boolean
* Symbol
* BigInt

## Estruturais

* Object
    - Array
    - Map
    - Set
    - Date
    _ ...
* Function

## Primitivo Estrutural / Structural Root Primitive

* Null
